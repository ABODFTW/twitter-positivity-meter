FAQ:

How to use it?

Type a twitter username in the field above, and you will get a score of the account positivity based on the top 10 words used by the account, and how many of the 10 words are natural, positive or negative.

How many tweets will be analysed?

By default it analyses the last 100 tweets.

What tech stack did you use?

Python Django framework in the backend, Materlize.css and some custom CSS for styling the frontend and Chart.js for the charts and textblob for words sentiment analysis and getting the tweets using getoldtweets3
